Lazy Hub Leaver
A lightweight and modern utility script for Roblox that provides a quick "Disconnect" button, allowing users to leave their current session with a single click.
Features
• Modern UI: Designed with a clean, minimal look using the Gotham font.
• Interactive: Built-in hover effects (background color transitions) for better user feedback.
• Lightweight: Low performance overhead with no external dependencies required.
• Customizable: Easily modify the button position, size, or colors directly in the script.
Usage
To use this script in your project, copy the code and place it into a LocalScript located within StarterPlayerScripts or your preferred execution environment.
